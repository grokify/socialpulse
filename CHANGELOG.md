## [v0.1.0] - 2026-03-30

### Highlights

- MkDocs-like static site generator for article discussion summaries with persona analysis and sentiment tracking
- Anti-hallucination verification tools to validate quotes and URLs against platform APIs

### Added

- Core types for discussion summaries: `Summary`, `Persona`, `Quote`, `Digest`, `Site` ([`a2a9600`](https://github.com/grokify/socialpulse/commit/a2a9600))
- JSON schema generation from Go types with embedded validation ([`055db7f`](https://github.com/grokify/socialpulse/commit/055db7f))
- `textutil` package for HTML stripping, quote removal, keyword extraction, and text scoring ([`88c8ee7`](https://github.com/grokify/socialpulse/commit/88c8ee7))
- Static site builder with data dashboard theme and ECharts visualizations ([`506fa0c`](https://github.com/grokify/socialpulse/commit/506fa0c))
- CLI commands: `new`, `serve`, `build`, `validate`, `verify`, `fetch-quotes`, `gh-deploy` ([`0fdfa05`](https://github.com/grokify/socialpulse/commit/0fdfa05))
- Development server with live reload via file watching ([`0fdfa05`](https://github.com/grokify/socialpulse/commit/0fdfa05))
- HackerNews API integration for URL and quote verification ([`0fdfa05`](https://github.com/grokify/socialpulse/commit/0fdfa05))
- Persona matching algorithm using keyword analysis ([`0fdfa05`](https://github.com/grokify/socialpulse/commit/0fdfa05))
