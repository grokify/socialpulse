# v0.1.0

**Release Date:** 2026-03-30

## Highlights

- MkDocs-like static site generator for article discussion summaries with persona analysis and sentiment tracking
- Anti-hallucination verification tools to validate quotes and URLs against platform APIs

## Added

- Core types for discussion summaries: `Summary`, `Persona`, `Quote`, `Digest`, `Site`
- JSON schema generation from Go types with embedded validation
- `textutil` package for HTML stripping, quote removal, keyword extraction, and text scoring
- Static site builder with data dashboard theme and ECharts visualizations
- CLI commands: `new`, `serve`, `build`, `validate`, `verify`, `fetch-quotes`, `gh-deploy`
- Development server with live reload via file watching
- HackerNews API integration for URL and quote verification
- Persona matching algorithm using keyword analysis

## Fixed

- Resolve golangci-lint errors: errcheck, gosec, dupl

## Tests

- Example summary content for validation testing
- Comprehensive textutil tests for HTML stripping and keyword extraction

## Documentation

- Project README with installation, quick start, and CLI reference
- MkDocs documentation site with Material theme
- CLI command documentation for all 7 commands
- Content authoring guide and schema reference

## Infrastructure

- Project configuration with MIT license
- GitHub Actions workflows for CI, linting, and SAST
- Gitignore for generated site output and binaries

## Dependencies
